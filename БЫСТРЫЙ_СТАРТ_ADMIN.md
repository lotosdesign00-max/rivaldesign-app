## ⚡ ШАГ 1: Настройка Supabase (2 минуты)

### 1.1 Открой SQL Editor

1. Зайди на: https://tlzxcghfvgazkzaoawtj.supabase.co
2. Войди в свой проект
3. В левом меню нажми **SQL Editor**
4. Нажми **New Query**

### 1.2 Запусти схему

1. Открой файл `supabase/admin_schema.sql` из своего проекта
2. Скопируй **ВСЁ** содержимое (Ctrl+A → Ctrl+C)
3. Вставь в SQL Editor (Ctrl+V)
4. Нажми **Run** (или Ctrl+Enter)

### 1.3 Проверь результат

Должно появиться сообщение **"Success. No rows returned"** или список созданных таблиц.

**Созданные таблицы:**
- ✅ `gallery` — портфолио
- ✅ `courses` — курсы
- ✅ `services` — услуги
- ✅ `reviews` — отзывы
- ✅ `faq` — вопросы/ответы
- ✅ `home_stats` — статистика главной
- ✅ `home_socials` — соцсети

## 🔑 ШАГ 2: Добавь ключи в Vercel (1 минута)

### 2.1 Найди свои ключи Supabase

1. Открой: https://tlzxcghfvgazkzaoawtj.supabase.co
2. В левом меню нажми **Settings** (шестерёнка внизу)
3. Нажми **API**
4. Скопируй два ключа:
   - **`anon` `public`** → это `VITE_SUPABASE_ANON_KEY`
   - **`service_role` `secret`** → это `SUPABASE_SERVICE_KEY`

⚠️ **ВАЖНО:** `service_role` ключ — СЕКРЕТНЫЙ! Не показывай никому!

### 2.2 Добавь в Vercel

1. Открой: https://vercel.com/dashboard
2. Зайди в свой проект
3. Нажми **Settings** → **Environment Variables**
4. Добавь первую переменную:
   - **Name:** `SUPABASE_SERVICE_KEY`
   - **Value:** вставь `service_role` ключ
   - **Environment:** выбери все (Production, Preview, Development)
   - Нажми **Save**
5. Добавь вторую переменную:
   - **Name:** `SUPABASE_URL`
   - **Value:** `https://tlzxcghfvgazkzaoawtj.supabase.co`
   - **Environment:** выбери все
   - Нажми **Save**

## 🔄 ШАГ 3: Перезапусти деплой (1 минута)

1. Зайди в Vercel Dashboard → свой проект

6. Подожди 1-2 минуты пока пересоберётся

## 📱 ШАГ 5: Создай Telegram Mini App (по желанию)

### 5.1 Создай бота

1. Открой Telegram
2. Найди **@BotFather**
3. Напиши `/newbot`
4. Следуй инструкциям:
   - Дай имя: `Rival Admin`
   - Дай username: `rival_admin_bot` (или любой другой)
5. **Сохрани токен** бота (пригодится позже)

### 5.2 Создай Mini App

1. Напиши @BotFather: `/newapp`
2. Выбери своего бота
3. Загрузи иконку (картинку 512x512)
4. Дай название: `Rival Admin Panel`
5. Дай описание: `Админ-панель для управления контентом`
6. Короткое описание: `Admin`
7. Когда попросит **URL**, вставь:
   ```
   https://твой-домен.vercel.app/admin
   ```

✅ **ГОТОВО!** Теперь админка доступна через Telegram!

### 5.3 Как открыть админку из Telegram

1. Открой своего бота
2. Нажми кнопку **Menu** (слева от поля ввода)
3. Или нажми на ссылку Mini App в сообщении от BotFather
4. Откроется админ-панель прямо в Telegram!

## 🎯 КАК ПОЛЬЗОВАТЬСЯ АДМИНКОЙ

### 📊 Dashboard (Главная)
- Статистика: заказы, доход, клиенты
- Последние заказы
- Последние платежи

### 📦 Портфолио
1. Нажми **📦 Портфолио** в меню слева
2. Нажми **➕ Добавить**
3. Заполни форму:
   - **Название** — имя работы
   - **Категория** — выбери из списка
   - **Описание** — расскажи о работе
   - **URL картинки** — ссылка на фото
   - **Теги** — добавь теги
   - **Популярное** — отметь если хочешь выделить
4. Нажми **➕ Добавить**

### 📚 Курсы
1. Нажми **📚 Курсы**
2. Нажми **➕ Добавить**
3. Заполни:
   - Название, категория, описание
   - Уровень (Новичок/Средний/Продвинутый)
   - Длительность, количество уроков
   - Цена, рейтинг
   - Темы курса (список)
4. Нажми **Добавить**

### 💼 Услуги
1. Нажми **💼 Услуги**
2. Нажми **➕ Добавить**
3. Заполни название, цену, описание
4. Нажми **Добавить**

### ⭐ Отзывы
1. Нажми **⭐ Отзывы**
2. Нажми **➕ Добавить**
3. Заполни имя, рейтинг, текст
4. Нажми **Добавить**

### ❓ FAQ
1. Нажми **❓ FAQ**
2. Нажми **➕ Добавить**
3. Напиши вопрос и ответ
4. Нажми **Добавить**

### 🏠 Главная
- Редактирование статистики (цифры на главной)
- Редактирование соцсетей (ссылки)

### 📝 Входящие заказы
1. Нажми **📝 Входящие**
2. Выбери заказ из списка слева
3. Измени статус (выбор из списка)

2. Видишь список всех клиентов
3. Измени баланс (введи сумму → нажми 💾 Сохранить)

```
Ты в админке добавляешь работу
        ↓
```

Когда ты зальёшь новую версию кода на GitHub → Vercel пересоберёт сайт → сайт всё равно покажет ВСЕ работы из базы данных!

## ⚠️ Если что-то не работает

### "Missing SUPABASE_SERVICE_KEY"
- Проверь что добавил ключ в Vercel Environment Variables
- Перезапусти деплой (Redeploy)

### "Таблицы не найдены"
- Убедись что запустил `supabase/admin_schema.sql` в Supabase SQL Editor
- Проверь что нет ошибок в консоли

### "Нет данных в админке"
- Это нормально! База новая и пустая
- Добавь контент через админку

### "Admin API request failed"
- Проверь логи в Vercel Dashboard → Deployments → кликни на деплой → Logs
- Убедись что serverless функции работают

### Админка не открывается
- Проверь URL: `https://твой-домен.vercel.app/admin`
- Проверь консоль браузера (F12) на ошибки
