# Быстрый старт админки

## Что открыть

Локально:
- `http://localhost:5173/admin/`

После деплоя:
- `https://your-domain/admin`

## Что нужно перед стартом

1. Заполнить `.env`
2. Применить `supabase/admin_live_schema.sql`
3. Запустить `npm install`
4. Запустить `npm run dev`

## Важные переменные

```env
SUPABASE_URL=
SUPABASE_SECRET_KEY=
CRYPTOPAY_API_TOKEN=
```

## Что умеет админка

- Inbox по заказам и запросам
- платежи и статусы
- клиенты и баланс
- редактирование услуг
- редактирование отзывов и FAQ
- управление главным контентом

## Вход локально

Если Telegram-контекста нет, админка может работать через local admin режим в браузере.

## Если данные не приходят

Проверь:
- что схема Supabase применена
- что серверный ключ указан в `.env`
- что dev-сервер перезапущен после изменения `.env`
