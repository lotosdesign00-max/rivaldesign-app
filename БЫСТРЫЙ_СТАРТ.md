## 📁 Подготовка Google Drive

1. Создай папку "Design Pack"
2. Создай подпапки: `Fonts`, `Textures`, `Brushes`, `Templates`, `Mockups`, `Graphics`
3. Загрузи файлы
4. **Сделай папку публичной:**
   - Правой кнопкой → "Настроить доступ"
   - "Доступ имеют все в интернете"
   - Роль: "Читатель"
5. Скопируй ID из URL:
   ```
   https://drive.google.com/drive/folders/1ABC123XYZ...
                                           ^^^^^^^^^^
                                           Это ID
   ```
