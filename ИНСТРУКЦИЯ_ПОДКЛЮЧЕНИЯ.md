## 🚀 Способ 1: Быстрый тест (не изменяя App.jsx)

### Создай тестовую страницу:

1. Создай файл `test-freepack.html` в корне проекта:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Test Free Pack</title>
</head>
<body>
  <div id="root"></div>
  <script type="module">
    import FreePackTabNew from './src/components/FreePackTabNew.jsx';
    import { createRoot } from 'react-dom/client';
    import { createElement } from 'react';
    
    const th = {
      card: "#1a1a1a",
      surface: "#0f0f0f",
      accent: "#00ff88",
      accentB: "#0088ff",
      text: "#fff",
      sub: "#888",
      border: "#333",
      grad: "linear-gradient(135deg, #00ff88, #0088ff)",
      glow: "#00ff8844",
      shadow: "0 4px 12px rgba(0,0,0,0.5)",
      btnTxt: "#000"
    };
    
    createRoot(document.getElementById('root')).render(
      createElement(FreePackTabNew, { th, lang: 'ru', SFX: {} })
    );
  </script>
</body>
</html>
```

2. Открой в браузере: `http://localhost:5173/test-freepack.html`

## 🔧 Способ 2: Подключить в App.jsx

### Вариант A: Заменить старый FreePackTab

1. Открой `src/App.jsx`

2. Найди строку с импортом (в самом начале):
```javascript
import { useState, useEffect, useRef } from "react";
```

3. Добавь после неё:
```javascript
import FreePackTabNew from './components/FreePackTabNew.jsx';
```

4. Найди в коде где рендерится `<FreePackTab ... />` (примерно строка 2400+)

5. Замени на:
```javascript
<FreePackTabNew th={th} lang={lang} SFX={SFX} />
```

### Вариант B: Добавить как новую вкладку

1. Импортируй компонент (см. выше)

2. Найди массив вкладок `TABS` в App.jsx

3. Добавь новую вкладку:
```javascript
{ 
  id: "freepack_new", 
  icon: "🎁", 
  label: { ru: "Новый Пак", en: "New Pack" } 
}
```

4. В switch/case добавь:
```javascript
case "freepack_new":
  return <FreePackTabNew th={th} lang={lang} SFX={SFX} />;
```

## 📝 Google Drive настройка

1. Создай публичную папку на Google Drive
2. Для каждого файла получи прямую ссылку:
   - Правой кнопкой → "Получить ссылку"
   - "Доступ имеют все в интернете"
   - Скопируй ID файла из URL
   - Формат: `https://drive.google.com/uc?export=download&id=FILE_ID`

## 🐛 Решение проблем

### Компонент не отображается
- Проверь что импорт правильный
- Проверь консоль браузера (F12) на ошибки

### Файлы не скачиваются  
- Проверь что `downloadUrl` правильный
- Проверь что файл публичный в Google Drive

### Подписка не срабатывает
- Сейчас задержка 2 секунды (имитация)
- Для реальной проверки нужен Telegram бот
